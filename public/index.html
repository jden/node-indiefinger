indiefinger

a proxy for webfinger

indiefinger provides a webfinger-compatible endpoint which you can use to get information about URLs and email addresses. If the given target resource operates its own webfinger service, this acts as a transparent proxy. If not, we'll attempt to discover links from that resource by parsing HTTP and HTML links.

make your first indiefinger request!

http://indiefinger.org/.well-known/webfinger?resource=

use indiefinger from the command line

since indiefinger is built on webfinger, you can run it with curl

$ curl http://indiefinger.org/.well-known/webfinger?resource=http://indiefinger.org

or, with node.js installed, you can download and run indiefinger as a command line tool.

$ npm install -g indiefinger-cli

now, you can use indiefinger (installed as ifinger):

$ ifinger indiefinger.org
