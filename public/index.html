indiefinger

a proxy for webfinger

indiefinger provides a webfinger-compatible endpoint which you can use to get information about URLs and email addresses. If the given target resource operates its own webfinger service, this acts as a transparent proxy. If not, we'll attempt to discover links from that resource by parsing HTTP and HTML links.
